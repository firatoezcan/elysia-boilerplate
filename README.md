| Command      | Script                  | Description                           |
| ------------ | ----------------------- | ------------------------------------- |
| `bun dev`    | bun --watch src/main.ts | Starts the development server         |
| `bun bundle` | bun build.ts            | Bundles the server into a single file |
| `bun start`  | bun dist/main.ts        | Runs the production server            |
